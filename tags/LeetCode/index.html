      • 第 i-1 天持有股票：dp[i][1] = prices[i] + dp[i-1][0] - fee
    • 两种情况取最大值

  3. dp 数组初始化： dp[0][0] = -prices[0]，dp[0][1] = 0

简单推理

  1. 假设字符串 s 使用多个重复子串（最小重复单位）构成，重复出现的子字符串长度是 x，有 s.length = n * x。

  3. 若 nx % (n - m)x = 0 => n % (n - m) = 0，就可以判定有重复出现的子字符串
